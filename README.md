# Progressive-Web-App
![alt text](dziura.jpg)

Aplikacja typu Progressive Web App, która służy do udostępniania miejskich zgłoszeń takich jak: dziura w drodze, źle zaparkowany pojazd. 

Użytkownik przesyła po wejściu do aplikacji ma możliwość zrobienia zdjęcia zgłoszenia oraz naciska przycisk udostępniania swojej lokalizacji. Po wykonaniu poprzednich kroków użytkownik udostępnia zdjęcie oraz lokalizację poprzez Web Share Api.
